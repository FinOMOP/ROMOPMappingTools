# Plot Table Vocabulary Status

This function plots the vocabulary status for a specific source
vocabulary ID.

## Usage

``` r
plotTableVocabularyStatus(mapping_status, source_vocabulary_id)
```

## Arguments

- mapping_status:

  The mapping status data containing code counts.

- source_vocabulary_id:

  The source vocabulary ID for which to plot the status.

## Value

A reactive table displaying the vocabulary status.
