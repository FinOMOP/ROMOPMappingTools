# Get Usagi summary for a vocabulary

Get Usagi summary for a vocabulary

## Usage

``` r
.getUsagiSummaryForVocabulary(pathToVocabularyFolder, sourceVocabularyId)
```

## Arguments

- pathToVocabularyFolder:

  Path to vocabulary folder

- sourceVocabularyId:

  Source vocabulary ID

## Value

Usagi summary tibble
