# Plot Usagi file table

Plot Usagi file table

## Usage

``` r
.plotTableForUsagiFile(
  usagiTibble,
  colors = list(flagged = "#EC6173", unchecked = "#F1AE4A", approved = "#51A350", inexact
    = "#AAAAAA")
)
```

## Arguments

- usagiTibble:

  Usagi tibble

- colors:

  List of colors

## Value

A reactable table object
