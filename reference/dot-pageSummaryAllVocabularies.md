# Render summary page for all vocabularies

Render summary page for all vocabularies

## Usage

``` r
.pageSummaryAllVocabularies(summaryAllVocabularies, outputFolderPath)
```

## Arguments

- summaryAllVocabularies:

  Tibble with summary for all vocabularies

- outputFolderPath:

  Output folder path

## Value

Path to the output HTML file
