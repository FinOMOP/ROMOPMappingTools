# Get database summary for a vocabulary

Get database summary for a vocabulary

## Usage

``` r
.getDatabaseSummaryForVocabulary(
  connectionDetails,
  vocabularyDatabaseSchema,
  targetVocabularyIds
)
```

## Arguments

- connectionDetails:

  Connection details

- vocabularyDatabaseSchema:

  Vocabulary database schema

- targetVocabularyIds:

  Target vocabulary IDs

## Value

Database summary tibble
