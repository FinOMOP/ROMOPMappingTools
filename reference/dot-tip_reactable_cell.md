# Tooltip for reactable cell

Tooltip for reactable cell

## Usage

``` r
.tip_reactable_cell(
  string_values,
  row_names = NULL,
  unit_names = NULL,
  only_percent = FALSE
)
```

## Arguments

- string_values:

  String with values separated by '-'

- row_names:

  Row names for tooltip

- unit_names:

  Unit names for tooltip

- only_percent:

  Logical, show only percent

## Value

A shiny div with tippy tooltip
