# Plot summary table for Usagi file

Plot summary table for Usagi file

## Usage

``` r
.plotSummaryTableForUsagiFile(
  usagiTibble,
  colors = list(flagged = "#EC6173", unchecked = "#F1AE4A", approved = "#51A350", inexact
    = "#AAAAAA")
)
```

## Arguments

- usagiTibble:

  Usagi tibble

- colors:

  List of colors

## Value

A reactable table object
